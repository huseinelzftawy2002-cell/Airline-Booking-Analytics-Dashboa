# ✈️ Airline Booking & Passenger Analysis Dashboard (Excel)

## 📌 Project Overview

This project presents an interactive **Airline Booking Dashboard** built entirely in **Microsoft Excel**. The dashboard provides a comprehensive analysis of booking performance, passenger behavior, and sales channels, enabling data-driven decision-making.

The goal of this project is to transform raw airline data into meaningful insights through clean visualization and intuitive dashboard design.

---

## 🎯 Objectives

* Analyze booking completion rates and passenger distribution
* Identify top booking origins and travel patterns
* Evaluate sales channel performance (Internet vs Mobile)
* Understand customer behavior regarding add-on services (baggage, seats, meals)
* Discover trends across days of the week and flight hours

---

## 📊 Key Insights

* **Total Bookings:** 50,000
* **Total Passengers:** 79,562
* **Booking Completion Rate:** ~85%
* **Top Booking Origins:** Australia, Malaysia, South Korea
* **Peak Travel Days:** Monday and Tuesday
* **Dominant Sales Channel:** Internet (~89%)

### ✨ Ancillary Services

* **Baggage Purchase:** 67%
* **Seat Selection:** 70%
* **Meal Purchase:** 57%

### ⏱️ Operational Trends

* Passenger activity peaks in mid-range flight durations
* Clear variation in travel demand across weekdays
